# 前言

电脑是高考结束之后买的华硕天选 2，R7 5800H + RTX 3060，7499入手，就从现在看来当时买还是不太亏的，便宜有现货，就是性能丐一点，没有独显直连，屏幕不是特别好，但是我也看不出来，无所谓。我记得最开始用应该是没什么问题的，后面重装过几次系统，Windows 和 Ubuntu 都重装过，可能是因此发生了一些变化导致现在看 b 站直播都能卡。

> 目前是用了一根TypeC转dp的线外接显示屏，这样是可以独显直连的，让核显睡觉去吧

# 原因分析

![2.png](https://s2.loli.net/2023/08/19/bkAcDEraPuIH4iW.png)

习惯性打开任务管理器看一眼，cpu 和内存占用很正常，百分之五十以下，显卡是两张，一张 RTX 3060，6G **专用 GPU 内存**，一张 AMD 芯片自带核显，512MB **专用 GPU 内存**。经后面分析，就是这个 GPU 的缓存被大量占用导致电脑卡的不行，上面是打开 tim wallpaper 火绒 clash 四个应用的内存占用，99 MB，下面是再加上 B 站直播，436MB，已经卡死了，鼠标卡的非常难以操作，**已经到了不能正常使用的地步**

![6.png](https://s2.loli.net/2023/08/19/WDjaHEyXoQJsG9q.png)

但 3060 基本以 0 的占用在呼呼睡大觉，麻了。似乎只有在打游戏的时候才会调用独显。

![5.png](https://s2.loli.net/2023/08/19/nLiMV5PuCKSY89a.png)

# 问题解决

还是搜索的魅力，我换了很多关键词，`华硕天选2优化` `提升显卡性能` `独显直连`等等。首先华硕天选 2 是不支持独显直连的，只能采用混合模式或者核显模式，混合就是让Windows自行决定用什么显卡，那后面就想到是不是能让谷歌浏览器只使用独显就好了，再经理一番搜索之后终于，找到[这篇文章](https://www.bilibili.com/read/cv6266330?spm_id_from=333.999.0.0)。里面的几种方法建议都操作一下。我的电脑起作用的是下面这个，非常有用。调整完之后看 B 站直播开原画，核显占用 228 MB/512 MB，独显0.2/6 GB。

![3.png](https://s2.loli.net/2023/08/19/Nq7L6AU8fpkJs1S.png)

还有一点是 Windows 的任务管理器左边的百分比显示一般是指 3D 一栏，专用 GPU 内存利用率要在右边看。

![4.png](https://s2.loli.net/2023/08/19/9yrxlpOtASJQu1P.png)

# 总结

感觉这个问题解决的难点在于如何搜索，在不能独显直连的限制下是否能想到指定某软件使用独显，否则难以搜索出对应的教程。我搜索了两三天才解决好😭

> 不过我看评论区也有说切换之后更卡的，猜测可能是显卡不支持 B 站默认 av1 视频的解码，可以尝试在右下角小齿轮-更多播放设置-更改播放策略试试（未验证）
>
> 此外提一点是华硕天选 2的外接显示屏独显直连需要使用 type-c 转 DP 的线，转 HDMI 似乎也可以。就是连笔记本不能连 HDMI 口，不然还是核显，确实是十分抽象的。

“Windows将双显卡的调用策略放到了系统级的设置中，于是导致Nvidia控制面板这一软件层面的设置不起作用了”

"windows10 2004， Chrome 83 亲测有效。刚重装了系统，以前都可以的，很可能是windows最新版的锅，真的时不时给你整些幺蛾子出来"

评论区有人这么说，感觉倒很合理，解释了为什么一开始我用着没问题，现在出问题了，因为Windows有更新我都是直接更的。
