# 常用命令行指令

```
# 用adb查看是否链接上设备
adb devices
# 正常系统时重启至fastboot模式
adb reboot bootloader
# 正常系统时重启至recovery模式
adb reboot recovery
----------------------------------
# 用fastboot查看是否链接上设备
fastboot devices
# 刷入boot镜像
fastboot flash boot boot.img
# bootloader模式下重启至系统
fastboot reboot
```

* 小米备份数据 [b站视频教程](https://www.bilibili.com/video/BV1MevWehEoE)

* 刷官改包或者老版本系统直接用[miflash](https://miuiver.com/miflash/)直接刷就行，最开始我先刷了一位佬做的澎湃os2，但是发现我找的up主做的第三方主题基本还没适配澎湃2，又刷回了1，然后发现有些软件不兼容，最后还是刷了MIUI12.5，真是应验了那句话，愿你刷机半生，归来仍是MIUI哈哈哈

* 给recover分区刷入twrp系统，不同手机刷入方法不同，具体可参考[这个视频](https://www.bilibili.com/video/BV1RD4y1J7Ku)

  米11指令为`fastboot boot twrp-3.7.0_11-0-venus.img`

  自动进入twrp

  高级→刷入当前的TWRP→确认（如果有清除dalvik选项就清除一下）

  > 如果使用澎湃os2显示挂载失败 无法挂载/data/media/TWRP
  >
  > 这个twrp镜像无法刷入，请在酷安找到一个更新的米11的twrp镜像

* [获取root权限方法一  修补boot镜像后刷入boot](https://magiskcn.com/)

  在TWRP中刷入magisk模块。把magisk后缀改成zip放入手机，在TWRP中点击安装，选择zip文件即可，重启后在手机上安装apk即完成刷入

> ```
> level=fatal msg="Parse config error: rules[512] [GEOIP,CN,DIRECT] error: can't download MMDB: context deadline exceeded"
> ```
>
> 可以前往[该仓库](https://github.com/MetaCubeX/meta-rules-dat)下载 `geoip.metadb`，并将其上传到与 config.yaml 相同的目录内再重新运行容器即可
>
> 按照`/data/adb/box/clash/config.yaml`的格式填入连接、修改规则即可，在magisk中启动模块，代理就成功启动了
>
> 访问`http://127.0.0.1:9090/ui`即可管理后台，修改节点了

# FL

* 安装软件`Fake Host`、`Fake Location_1.2.1.8`、`某软件`

* 安装模块`XccVip-1.23`和`隐藏应用列表HMA-V3.3`，其实也就是`apk`，`XccVip`安装后不会显示在桌面上

* `lsp`中启用两个模块，`XccVip`勾选`fake host`和`fake location`

* 重启

* 隐藏应用列表->应用管理->`某软件`和Fake location启用隐藏

  隐藏应用列表->模块管理->创建黑名单模板->不可见勾选Fake location->应用于`某软件`->写一个模板名

* 启动Fake Host、Fake location（位置模拟/线路模拟）和`某软件`，完成！
